Использованная СУБД: MySQL 5.7
Скрипт для БД находится в файле myshopdb.txt


Порядок использования:
1. Добавить категории товара, нажав на кнопку "Manage categories" (см. скриншот screen1)
2. Добавить товары, нажав на кнопку "Add new product"
3. Для поиска ввести в форму поиска начало имени или категории, нажать кнопку "Show results".
